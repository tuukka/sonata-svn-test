Support

The project is hosted at https://developer.berlios.de/projects/sonata/.
There you can file bug reports and feature requests, etc.

There are also a couple of mailing lists:
   sonata-users - Questions, comments, suggestions
   sonata-announce - Release announcements, low volume
   sonata-translations - For translators

And there is an IRC channel too:
   #sonata @ irc.freenode.net
